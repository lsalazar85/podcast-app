# Podcasts App

Podcasts app development with React.js and Next.js


## Develop Mode Run
```
    yarn install
    yarn run dev
```

```
    npm install
    npm run dev
```

Starts on `http://localhost:3000`


## Production Build Mode Run
```
    yarn install
    yarn run build
```

```
    npm install
    npm run build
```

## Development technologies and details
* React.js
* Next.js
* CSS3
* HTML5
* Responsive Design
* CSS Grid
